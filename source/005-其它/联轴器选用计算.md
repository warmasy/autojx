# 联轴器选用计算

## 计算表格

计算公式

```
T联=T电机*k1*k2*k3
```

| 负载系数k1 |      |
| ---------- | ---- |
| 恒负载     | 1.0  |
| 小变动负载 | 1.2  |
| 常变动负载 | 1.7  |
| 大变动负载 | 2.1  |

| 运转时间系数k2 |      |      |
| -------------- | ---- | ---- |
| 每天运转时间   |      |      |
| ≤2h            | 0.07 |      |
| ≤4h            | 0.85 |      |
| ≤8h            | 1.00 |      |
| ≤16h           | 1.18 |      |
| ≤24h           | 1.28 |      |

| 启动、停止频繁系数k3 |      |      |
| -------------------- | ---- | ---- |
| 每小时启停系数       |      |      |
| ≤10次1.0             | 1.0  |      |
| ≤30次1.1             | 1.1  |      |
| ≤60次                | 1.2  |      |
| ≤120次               | 1.5  |      |
| ≤240次               | 2.0  |      |
